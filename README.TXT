-The source code "yo.php" allows you to label and obtain the coordinates
 of your objects in the images compatible with Opencv library.

-the project contains 2 files "style.css" and "yo.php"
-the code does not use any library, written in html, css, pure javascript and php
-to use the project you just need to have a web server and a file in htdoc which
 contains the images you want to label.
-the code is developed in linux (xubunto), if you work on linux you must enter
 in the first field a relative path example: "./images_files/", 
 if you work on windows you may find errors in this case check 
 the php part in the source code.
